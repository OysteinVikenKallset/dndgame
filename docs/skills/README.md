# Skills

Denne mappen beskriver ferdigheter (skills) som AI-agenter og utviklere skal følge i prosjektet.

## Formål

- Standardisere arbeidsmåte på tvers av agenter
- Gjøre beslutninger og praksis tydelige
- Redusere variasjon i kvalitet og stil

## Hvordan bruke skills

- Les relevant skill før du starter en oppgave
- Følg sjekklisten i skill-dokumentet under arbeid
- Oppdater skillen når teamet forbedrer praksisen
- Bruk `skill-template.skill.md` som obligatorisk utgangspunkt for nye skills

## Filnavnkonvensjon (nye skills)

- Skills navngis som `*.skill.md`
