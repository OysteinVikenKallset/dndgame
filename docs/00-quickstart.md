# 00 Quickstart

Last reviewed: 2026-02-25

## Purpose

Rask oppstart for utviklere og AI-agenter.

## Start

1. Installer dependencies
2. Kjør `npm run check`
3. Les `docs/README.md` for decision tree
4. Les `docs/skills/ai-agent-doc-compliance.skill.md` (gjelder alltid for AI)
5. Les relevante skills i `docs/skills/` før kodeendring
6. Bruk `Compliance Summary` i PR-mal ved levering
7. Kjør `npm run docs:guard` ved behov for å verifisere docs-regler lokalt

## Før første PR

- Følg TDD
- Oppdater docs ved kodeendring
- Oppdater ADR ved varig arkitekturvalg
- Verifiser med `npm run check`
