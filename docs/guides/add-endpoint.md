# Guide: Add Endpoint

Last reviewed: 2026-02-24

## Purpose

Legge til nytt HTTP-endpoint med riktig lagdeling og docs-sync.

## Steps

1. Les relevante skills i `docs/skills/`
2. Skriv test først (RED)
3. Implementer minimal løsning (GREEN)
4. Refaktorer trygt
5. Oppdater API-docs og relevante policies
6. Kjør `npm run check`

## Checklist

- [ ] Request/response dokumentert
- [ ] Feilstatuskoder dokumentert
- [ ] Docs-impact beskrevet i PR
